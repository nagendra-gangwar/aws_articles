# useful commands
* Use **free** with -h to check the memory details.
* use **vmstat** to displays  information about processes, memory,block IO, system, and CPU activity.
* use **top** command to display information about processes which are consuming more resources.
* use **htop** command to display information about processes which are consuming more resources.
* use **df** with -h to display disk mount and available space.
* use **ps** with aux option to display process details.
* use **du** /sch <dir name> for getting the detail of disk utilization.
